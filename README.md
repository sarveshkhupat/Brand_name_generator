# 🚀 Name Generator

> Generate unique usernames, brand names, and company name ideas in seconds using Python.

---

## ✨ Overview

Name Generator is a beginner-friendly Python project that creates creative name suggestions by combining predefined prefixes and suffixes with user input.

Whether you're looking for:

- 🎮 Gaming usernames
- 📱 Social media handles
- 🏢 Company names
- 🏷️ Brand names

this project can instantly generate multiple ideas.

---

## 🎯 Features

✅ Generate 10 random names instantly

✅ Simple and lightweight

✅ Easy to customize

✅ Beginner-friendly code structure

✅ No external libraries required

---

## 📸 Example

### Input

```text
Sarvesh
```

### Output

```text
1. techsarveshlabs
2. ultrasarveshworld
3. primesarveshzone
4. megasarveshx
5. neosarveshhub
```
